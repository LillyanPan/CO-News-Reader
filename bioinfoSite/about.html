Overview

This site is a database of high-quality and non-high-quality protein-protein interactions in S. cerevisiae. This was created through BTRY 4831 as a final project. To search for proteins only fill in one field.

Implementation

The site was implemented by intially populating the database with information provided in the files CervBinaryHQ.txt â€“ Protein interaction network of Saccharomyces cerevisiae and Yeastgeneinfo.txt: containing annotation information for the entire yeast genome using a python script. The database holds two tables, geneInfo and interP. GeneInfo holds information about the gene (with fields geneName, orf, entrezID, pubmed, evidCode, ht), while interP holds all interacting pairs. Both tables a quired to create a complete table when a user searches a gene. A description of the SQL table is included in table.pdf. If a user doesn't enter any information into the form and error appears and the user can try to search again. A similar error is given if nothing is found. Results are displayed in a table.

How to Use

1. Fill in one field and press 'Submit' 2. Links to the corresponding NCBI Entrez Gene database page are provided for each gene
